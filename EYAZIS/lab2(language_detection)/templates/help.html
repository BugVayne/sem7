Подробные инструкции
1. Загрузка файла
  • Перейдите на главную страницу
  • Нажмите "Выберите файл" и выберите HTML файл
  • Поддерживаются форматы: .html, .htm
  • Максимальный размер файла: 16MB
  • Нажмите "Анализировать файл" для начала обработки
2. Просмотр результатов
  • После обработки откроется страница результатов
  • Просмотрите сводную информацию по всем методам
  • Изучите детальные результаты для каждого метода
  • Обратите внимание на уровень уверенности и время обработки
3. Действия с результатами
  • Просмотреть HTML - открыть оригинальный файл в новой вкладке
  • Скачать файл - загрузить исходный HTML файл
  • Сохранить результаты - скачать результаты в JSON формате
  • Распечатать - создать версию для печати
  • В историю - перейти к списку всех анализов
4. История анализов
  • Все анализы автоматически сохраняются
  • Просматривайте историю на соответствующей странице
  • Возвращайтесь к результатам предыдущих анализов
  • Скачивайте файлы и результаты прошлых анализов
Методы определения языка
Алфавитный метод

Анализ на основе символов алфавита и их частотности в различных языках

N-gram метод

Анализ последовательностей символов (N-грамм) для повышения точности

Нейросетевой метод

Использование нейронных сетей для сложных случаев определения языка
